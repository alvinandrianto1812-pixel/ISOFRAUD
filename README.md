# ISOFRAUD

**Fraud Detection System using Isolation Forest ML**

Full-stack web application for detecting fraudulent financial
transactions using unsupervised machine learning.

## Tech Stack

- **ML Model**: Isolation Forest (Fraud Detection)
- **Backend**: Python · Flask
- **Frontend**: HTML · CSS · Jinja2
- **Database**: Supabase
- **Notebook**: Jupyter (EDA + Model Training)

## Features

- Complete ML pipeline: EDA → training → hyperparameter
  tuning (1200+ trials) → threshold optimization
- Admin dashboard: dataset upload, model evaluation, user management
- User dashboard: transaction input, fraud detection result, history
- ROC Curve, Confusion Matrix, Precision-Recall visualization
